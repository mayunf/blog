# mayunf.github.io

## 安装依赖
```shell
npm install
```

## 使用

### 新建文章

```shell
hexo new page --path code/docker "Install Docker"
```

### 本地预览
```shell
hexo server
```

### 发布文章

```shell
hexo generate -d
```

**参考文档**
[https://hexo.io/docs/commands](https://hexo.io/docs/commands)
